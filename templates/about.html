Accountability: About This Data Set

This dataset contains information about the percent of each country's land that is covered by forest. The dataset show how forest cover for each country has changed annually between 1990 and 2015.

Where did this data come from?

This dataset comes from the World Bank and the Food and Agriculture Organization. It can be found at this link: https://data.worldbank.org/indicator/AG.LND.FRST.ZS
